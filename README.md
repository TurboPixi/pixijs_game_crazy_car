## 游戏规则
通过点击游戏界面中的左右按钮，或通过键盘的的左右键点击，进行对障碍物的躲避，必要的时候可以通过在应急车道上进行躲避，但会消耗总分。撞击金币得到相应的总分加成，获得的总分越多即排名越靠前。

## 游戏介绍
**疯狂汽车** 是使用 **pixi.js** 框架，进行开发的一款H5小游戏。

## 游戏注意事项
由于没有做过多的适配兼容，所以请调整到 宽375px * 高750px 的界面上预览最佳。
